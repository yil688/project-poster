# BUSINFO 703 电商客户分群与运营策略分析项目

该项目基于电商客户交易和履置体验数据，使用 R 语言进行数据处理和无益纪审分群，通过 Power BI 构建交互式可视化，支持精准营销与客户运营策略决策。

## 内容
- 用于数据清洗、特征工程和聚类分析的 R 脚本
- 提供业务冥见和决策支持的 Power BI 报告
